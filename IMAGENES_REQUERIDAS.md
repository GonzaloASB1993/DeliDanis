## Instrucciones

### Opción 1: Usar tus propias fotos
1. Toma fotos profesionales de tus productos
2. Edítalas con buena iluminación
3. Guárdalas en `/public/images/` con los nombres exactos mencionados arriba

### Opción 2: Usar imágenes temporales (placeholder)
Mientras consigues las fotos reales, puedes usar servicios gratuitos:
- **Unsplash**: https://unsplash.com/ (busca: "cake", "cocktails", "pastries")
- **Pexels**: https://pexels.com/
- **Pixabay**: https://pixabay.com/

### Opción 3: Usar placeholders hasta tener fotos reales
Puedes usar servicios como:
- https://placehold.co/800x600
- https://via.placeholder.com/800x600

## Nota Importante
Para una mejor experiencia de usuario, te recomiendo:
1. Usar fotos REALES de tus productos (genera más confianza)
2. Mantener un estilo visual consistente
3. Optimizar las imágenes para web (comprimirlas sin perder calidad)
4. Usar formato WebP cuando sea posible para mejor rendimiento
